# BlogApp

이 프로젝트는 Spring Boot와 관련 기술 스택을 사용하여 개발된 간단한 블로그 애플리케이션입니다. 
사용자 인증, 게시글 관리 등 필수적인 웹 애플리케이션 기능을 포함하고 있습니다.

## ✨ 주요 기능

- **사용자 관리**: 회원가입 및 로그인 기능
- **인증 시스템**: JWT(JSON Web Token)를 이용한 안전한 인증 처리
- **게시글 관리**: 블로그 게시글 생성(Create), 조회(Read), 수정(Update), 삭제(Delete) (CRUD)
- **RESTful API**: 클라이언트와의 통신을 위한 API 엔드포인트 제공
- **서버 사이드 렌더링**: Thymeleaf를 사용한 동적인 웹 페이지 렌더링

## 🛠️ 기술 스택

- **Backend**: Java, Spring Boot
- **Security**: Spring Security, JWT
- **Database**: Spring Data JPA, MySQL
- **Build Tool**: Gradle
- **Frontend**: Thymeleaf, HTML, CSS, JavaScript
